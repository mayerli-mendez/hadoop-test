## Descripción del ejercicio

Este proyecto corresponde al taller **Hadoop - Prueba de Velocidad, Variedad y Visualización**, desarrollado como parte del tema **Ecosistema Hadoop y Big Data**.

El objetivo del ejercicio fue aplicar conceptos básicos de Big Data mediante el uso de **Python**, **MongoDB**, **Pandas** y **Matplotlib**. A través del programa se realizaron pruebas para insertar registros en una base de datos NoSQL, trabajar con documentos de diferentes estructuras y generar una visualización gráfica de los resultados obtenidos.

Durante la ejecución se probaron tres conceptos importantes:

* **Velocidad:** inserción de múltiples registros en MongoDB.
* **Variedad:** almacenamiento de documentos con diferentes estructuras.
* **Visualización:** creación de gráficos para interpretar los datos generados.

Como resultado, el programa generó el archivo **resultado_hadoop.png**, donde se muestran gráficos relacionados con el valor promedio por tipo de evento y la distribución de los eventos registrados.
